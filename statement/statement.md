# Problem Statement

Navigating a campus like VIT Bhopal can be really tough. Imagine you are a first-year student trying to find a lab.. You are a visitor trying to locate the auditorium. The campus is always growing. Old maps do not show the way to get from one place to another. This leads to a lot of wasted time and aimless wandering. There is a need for a tool that understands the campus layout. It should give you the efficient route.

My goal for this project is to build a Python-based Campus Navigation System. It will use AI search logic to make navigating the university easier.

Scope of the Project

The project currently focuses on getting the logic and campus mapping right.

# What it does now:

* I have turned spots like the Lab Complex, hostels and academic blocks into a searchable map of nodes.

* It uses AI algorithms to calculate the path between two locations.

* You just pick where you are and where you want to go from a list of campus landmarks.

* The system gives you a step-by-step path so you know exactly which turns to take.

# Where I want to take it

* Live GPS: I want to add real-time tracking. It will work like a campus Google Maps.

* Interactive Map: I plan to move from the console to a mobile app with a visual map.

* Traffic Awareness: I want to predict "rush hours" during class changes. It will suggest paths that avoid the crowds.

* Transport Modes: I plan to give routes depending on whether you are walking, cycling or waiting for the shuttle.

# Target Users

* Freshers and new students: They are still figuring out the layout. They need to get to class without getting lost.

* Parents and visitors: They visit for events. They are not familiar with where the different blocksre located.

* students: Other computing students want to see how AI search strategies work in a real-world scenario.

# Key Features

* Campus Database: A list of every major building and landmark, on campus.

* Route Optimizer: It finds the way while following actual campus paths.

* Error Check: A simple system that checks you are not trying to navigate to a place that does not exist.

* Quick Results: text-based instructions that tell you exactly how to get to your destination.

* Distance Stats: It tells you how you have to go. This helps you decide if you need to run or if you have time to walk.
